# Moja Galeria

Galeria fotograficzna zbudowana przy użyciu [Hugo](https://gohugo.io/) z motywem [hugo-theme-gallery](https://github.com/nicokaiser/hugo-theme-gallery).

## Galerie

- **Ostar 2025** – zdjęcia z regat Ostar 2025
- **USA 2025 Kalendarz** – wybrane zdjęcia z USA 2025

## Technologia

- Generator: Hugo v0.156+
- Motyw: hugo-theme-gallery
- Hosting: Cloudflare Pages

## Lokalne uruchomienie

```bash
hugo server
```

Strona dostępna pod adresem: http://localhost:1313
